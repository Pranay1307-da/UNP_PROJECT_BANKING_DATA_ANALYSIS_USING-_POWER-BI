# UNP_PROJECT_BANKING_DATA_ANALYSIS_USING-_POWER-BI
🏦 Banking Performance Analysis Dashboard

📌 Project Overview

I have done this project with collaboration of UNP, focusing on analyzing banking performance data to extract meaningful insights and improve decision-making.

This project presents an interactive Banking Performance Dashboard that analyzes key metrics such as transactions, customer behavior, branch performance, and financial trends using data visualization techniques.

---

🎯 Objectives

- Analyze overall banking performance using KPIs
- Understand customer behavior and demographics
- Evaluate branch-wise performance
- Monitor financial transactions and account activity
- Identify problems and provide actionable suggestions

---

📊 Dashboard Modules

🔹 KPI Overview

- Total Balance: $19.5K
- Average Balance: $1.95K
- Net Transactions: $4K
- Total Deposits & Withdrawals insights

🔹 Consumer Behavior Analysis

- Customer segmentation (Gender, Age group)
- Transaction patterns across demographics
- Geographic distribution of customers
- Customer ranking (Gold, Silver, Bronze)

🔹 Branch Operations Analysis

- Branch-wise transaction comparison
- Deposit performance by branch
- Average balance per branch
- Identification of top-performing branches

🔹 Financial Analysis

- Daily deposits vs withdrawals trends
- Net transactions by city
- Account type analysis (Savings vs Checking)

🔹 Accounts Overview

- Total accounts & transaction count
- Withdrawals by account holders
- Deposits distribution
- Balance by account type

---

⚠️ Problems Identified

- Lack of consolidated KPI monitoring
- Limited demographic insights
- Uneven branch performance
- High volatility in cash flow
- Uneven account utilization

---

💡 Suggestions & Solutions

- Implement KPI monitoring dashboards for real-time tracking
- Target high-potential age groups (26–40) with premium services
- Replicate strategies of top-performing branches
- Maintain liquidity buffers for stability
- Monitor high-withdrawal accounts to reduce churn

- Techniques: Data Cleaning, Data Visualization, KPI Analysis
